# 🌊 运行 Ocean Loop

## 快速开始（3步）

### 1️⃣ 启动服务器
```bash
python3 serve.py
```

### 2️⃣ 打开浏览器
- **调试页面**（推荐先测试）: http://localhost:8080/debug.html
- **完整版本**: http://localhost:8080/

### 3️⃣ 点击页面
- 点击鼠标创建粒子
- 应该立即看到白色圆圈出现

## 🎮 基本操作

| 操作 | 效果 |
|------|------|
| 点击 | 创建粒子 |
| 拖拽 | 修改粒子 |
| 空格 | 截图 |
| 退格 | 重置 |
| B | 收缩所有 |
| M | 静音/取消静音 |

## 🐛 如果有问题

1. **清除浏览器缓存**: Ctrl+Shift+R
2. **打开控制台**: F12 查看错误
3. **先测试调试页面**: debug.html 更简单
4. **查看修复文档**: FIXES.md
