# 🚀 Ocean Loop v2.0 快速开始指南

**欢迎使用 Ocean Loop v2.0！** 本指南将帮助你在 5 分钟内体验所有新功能。

---

## ⚡ 5 分钟快速体验

### 步骤 1：启动项目（30秒）

```bash
# 进入项目目录
cd /home/user/musiclab

# 启动服务器
python3 serve.py
```

然后在浏览器打开：**http://localhost:8080**

---

### 步骤 2：开启控制面板（10秒）

按键盘上的 **G 键**，右侧会出现控制面板。

你会看到 5 个分类：
- 🎨 Particle Settings
- 🌈 Visual Settings
- 🎵 Audio Settings
- ⚡ Physics Settings
- 🎮 Actions

---

### 步骤 3：尝试主题系统（30秒）

1. 展开 **🌈 Visual Settings**
2. 点击 **Theme** 下拉菜单
3. 依次尝试每个主题：
   - 🌊 Ocean（默认蓝色）
   - 🔥 Fire（火焰橙红）
   - 🌌 Galaxy（紫色星系）
   - 🌲 Forest（绿色森林）
   - 🌅 Sunset（日落暖色）
   - ⚫ Mono（黑白经典）

**看到颜色变化了吗？** 🎨

---

### 步骤 4：创建粒子并开启轨迹（1分钟）

1. **点击画布**创建几个粒子
2. 在控制面板找到 **Show Trails**，打开开关
3. 调整 **Trail Length** 滑块（试试 50）
4. **再次点击画布**创建更多粒子

**看到优雅的运动轨迹了吗？** ✨

---

### 步骤 5：调整粒子参数（1分钟）

展开 **🎨 Particle Settings**，试试这些调整：

1. **Max Particles** → 拖到 **80**
   - 现在可以创建更多粒子了！

2. **Speed** → 调到 **6**
   - 粒子移动更快了！

3. **Trail Length** → 调到 **80**
   - 轨迹更长更梦幻！

**点击画布创建大量粒子，感受变化！** 🎆

---

### 步骤 6：玩转音频控制（1分钟）

展开 **🎵 Audio Settings**：

1. **Volume** → 调到 **0.5**（提高音量）
2. **Reverb** → 调到 **0.8**（增加回声）
3. **Base Frequency** → 试试 **300**（更高音调）

**点击画布听听声音的变化！** 🎵

---

### 步骤 7：保存你的创作（30秒）

1. 创建一些满意的粒子效果
2. 展开 **🎮 Actions**
3. 点击 **💾 Save Scene**
4. 看到 "Scene saved!" 提示

现在：
- 点击 **🗑️ Clear All** 清空画布
- 点击 **📂 Load Scene** 加载回来
- **你的创作完美恢复了！** 💾

---

### 步骤 8：快捷操作（30秒）

在 **🎮 Actions** 分类中试试：

1. **🎲 Random Generate** → 随机生成粒子
2. **📉 Shrink All** → 所有粒子收缩
3. **📸 Screenshot** → 截图保存（也可以按空格键）

## 🐛 遇到问题？

### 控制面板不显示
- 按 **G** 键切换
- 刷新页面（Ctrl + Shift + R）
- 检查浏览器控制台（F12）

### 轨迹不显示
- 确保 **Show Trails** 已开启
- 粒子需要移动才能看到轨迹
- 增加 Speed 参数

### 粒子无法创建
- 检查是否达到 Max Particles 限制
- 尝试点击 Clear All 重置

### 没有声音
- 点击画布任意位置激活音频
- 检查 Volume 是否为 0
- 按 M 键检查是否静音

## 💡 小提示

1. **主题选择**
   - 暗色背景用 Ocean/Galaxy/Monochrome
   - 想要活力选 Fire/Sunset
   - 自然风格选 Forest

2. **性能优化**
   - 移动端建议 Max Particles < 30
   - 录制视频时关闭不必要的轨迹
   - 卡顿时降低粒子数量

3. **创作技巧**
   - 慢速度 + 长轨迹 = 优雅
   - 快速度 + 短轨迹 = 动感
   - 多粒子 + 无轨迹 = 混乱美

4. **音频体验**
   - 戴耳机效果更佳
   - 立体声平移需要双声道
   - 调整 Reverb 增加空间感
